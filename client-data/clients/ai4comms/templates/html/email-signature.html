INSTALLATION NOTES

Outlook / Apple Mail / Gmail: Copy the rendered signature block above (not the HTML source) and paste into the email client's signature settings.

HTML source: For programmatic deployment, copy the raw HTML between the <!-- /signature --> markers in the source of this page.

No images: All variants are text-only. Most email clients block external images by default, which would break image-based signatures for most recipients.

Fonts: Signatures fall back to Arial/Consolas gracefully. Syne and JetBrains Mono only render if the recipient has them installed — the fallback still maintains hierarchy.

Links: Replace personal details (name, title, email, phone) before deploying to team members.
